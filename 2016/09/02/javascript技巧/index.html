JavaScript常用技巧

02 Sep 2016

javascript常用技巧总结。
使用逻辑符号&&或者||进行条件判断
foo == 5 || doSomething()；等价于：if(foo != 5) doSomething();
还可以连级妙用：function a(){...}; var b = true; var c = b && a(); c && console.log(c)可以用控制c的boolean值来改变后续程序的执行走向。
转换boolean
…

undefined –未定义的值 undefined==undefined //true
boolean –true == 1//true false == 0 //true
string –!!string //string不为空就为true，string为空就为false 注意：console.log(Boolean(‘ ‘));//true
number –!!number //number为0时就为false，number不为0时就为true !!null // false
!!undefined // false
!!NaN // false
object–!!object //object为空或不为空都返回false

  • 值类型转换
    var ss= {0:'323',1:'324',length:2}；转换成array
    Array.prototype.slice.call(ss) //['323','324']
  • map()方法来遍历数组
    [1,2,3,4].map(function (val) { return val * val; });
  • 逗号操作符
    var a = 0;
    var b = (a++,99);
    console.log(a); //1
    console.log(b); //99

  • 避免变量全局污染
    var a = 23; console.log();等价于window.a = 23;
    可将window和document传为参数

  • 使用严格模式
    “use strict”； 好处是可以严格发现错误的地点。
