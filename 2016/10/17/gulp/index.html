gulp入门随即用

17 Oct 2016

直接进入主题：gulp是一个前端打包工具，主要功能就是移入要处理的文件，然后输出处理后的文件的过程。
gulp的安装：npm install gulp -g，再到文件根目录下安装：npm install gulp --save-dev;

主要API介绍：

gulp有五个方法：task ， run ， watch ， src ，和 dest。

读文件：

gulp.src():输入单个文件、数组文件和匹配模式，如：
单个文件：gulp.src('src/scss/index.scss'),多个文件：gulp.src(['src/js/index.js', 'src/js/layout.js'])，匹配模式：gulp.src('src/**/*.scss');

写文件：

gulp.dest('目录')，如：gulp.src(‘index.scss’).pipe(gulp.dest(‘输出目录’))

运行多个任务

gulp.run(),接收多个task任务名称，如：
gulp.run('testScss', 'js', 'index', 'image')，
run方法已弃用，你可以构建一个任务来执行这些任务即可，如：
gulp.task('allTask',['testScss', 'js', 'index', 'image']);

使用：

gulp其实就是将一个或多个文件复制或经过处理打包成一个文件或多个文件，且放到指定文件夹的过程。
项目文件根目录下创建配置gulp的gulpfile.js,可以写任务，一般一个task处理一种文件，用法：直接到命令行中输入gulp 任务名即可执行打包任务。
