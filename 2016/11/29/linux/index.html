Linux常用命令

Nov 29, 2016 in Testing

开发当中常用的Linux命令：
创建、删除、复制，查看、搜索等功能，
更多见详情…

  1. 列出文件夹下所有目录和文件：ls [选项]
    常用选项有：
    -a     列出所有隐藏的文件
-l     列出文件的详细信息
-t       根据文件的修改时间排序
-h     以容易理解的方式列出文件大小
  2. cd命令：
    cd [文件夹名称]
    cd / 进入系统根目录
    cd - 进入上次所在目录
    cd ~ 进入当前用户主目录
    cd .. 进入上一级目录
  3. 移动或重命名文件： mv [选项] 源文件 目标文件
    选项常用的： 
    -f: 如存在强制覆盖
-i: 询问是否覆盖
-t: 将多个源文件移动到统一目录下，目录参数在前，文件参数在后
如：mv a.js /template   
mv a.html b.html  
mv /individual/tblog test1.txt test2.txt test3.txt
  4. 复制到目标文件： cp [选项] 原文件 目标文件
    选项：
    -f 不询问强制复制
-a 将所有特性都复制过去
  5. 删除文件： rm [选项] 文件
    选项：
    -r   删除文件夹
-f   删除不提示
-i   删除提示
-v   详细显示删除进行的步骤
  6. 创建文件： touch [选项] 文件
    选项：
    -a    只修改存取时间
-m    只修改变动的时间
-r    使b时间和a相同   touch -r a b
-t    指定特定时间 touch -t 20161002.50 a.md
  7. 创建文件夹 mkdir命令： mkdir [选项] 目录
    选项：-p 如果目录不存在将自动建立好，如 individual/demo
    -v   每次创建都显示目录信息
-m   设定权限模式
  8. cat命令： cat [选项] [文件]
    选项：cat -n a.log > b.og 把a的内容加上行号输入到b中
    cat -ns a.log > b.og 把a的内容加上行号输入到b中，多行空行换成一行输入
tac a.log   将a的内容反向显示
  9. more命令：功能类似 cat ，cat 命令是将整个文件的内容从上到下显示在屏幕上。 more 命令会一页一页的显示，方便使用者逐页阅读，而最基本的指令就是按空白键（space）往下一页显示，按 b 键就会往回（back）一页显示，而且还有搜寻字串的功能 。more 命令从前向后读取文件，因此在启动时就加载整个文件。
    more [选项] 文件
    +n  从第几行开始显示
-n  定义屏幕大小为几行
+/pattern  搜寻该字符串，然后从改字符串前两行开始显示
-c  从顶部清屏在显示
-s  把连续的空行显示为一行
-u  把文中的下划线去掉
=   输出当前行的行号
q   退出more
空格键   向下滚一屏
b    返回一屏
  10. which命令：which 可执行的文件名称
  11. 其他常用命令：
    显示磁盘的使用情况：df
    显示时间命令：time date
    软链接建立：ln -s [源文件] [目标文件]
