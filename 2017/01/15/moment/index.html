moment的常见用法

15 Jan 2017


时间操作利器———momentJs。常见的用法总结。

格式化

number: 当前时间的毫秒数；
moment(number).format('YYYY-MM-DD HH:mm');

取值

  • 年： moment().year();
  • 月： moment().get(‘month’); // 0 to 11
  • 日： moment().get(‘date’);
  • 小时： moment().get(‘hour’);
  • 分： moment().get(‘minute’);
  • 秒： moment().get(‘second’);
  • 毫秒：moment().get(‘millisecond’);
    以上都写法和moment().month().moment.year()等价；
  • 星期几： moment().day()；// 0-6 moment().isoWeekday(); //1-7
  • 某个月有多少天：moment().daysInMonth();
  • 一周的第几天是哪一天： moment().isoWeekday(number).format(‘YYYY-MM-DD’)
  • 一年的第多少天：moment().dayOfYear(number)； moment().dayOfYear(number)设置为第多少天
  • 一年中的第几周： moment.week(); moment.week(number)设置为第几周
  • 一年中有多少周： moment().weeksInYear();
  • 季度： moment().quarter(); // Number moment().quarter(Number);

赋值

  • 年：moment().set(‘year’, 2013);
  • 月： moment().set(‘month’, 3); // April
  • 日： moment().set(‘date’, 1);
  • 小时： moment().set(‘hour’, 13);
  • 分钟：moment().set(‘minute’, 20);
  • 秒 moment().set(‘second’, 30);
  • 毫秒 moment().set(‘millisecond’, 123);
  • 对象的形式赋值： moment().set({‘year’: 2013, ‘month’: 3});

运算

  • 加法：moment().add(number, ‘days/d/month/year/week….’)
  • 减法：moment.subtract(number, ‘day/d/month/year/week….’)
  • 开始时间：
    • 某一周的第一天： moment().startOf(‘isoWeek’).format(‘YYYY-MM-DD’)
    • 某一月的第一天： moment().startOf(‘month’).format(‘YYYY-MM-DD’)
      …
  • 结束时间：
    • 某一周的最后一天： moment().endOf(‘isoWeek’).format(‘YYYY-MM-DD’)
    • 某一月的最后一天： moment().endOf(‘month’).format(‘YYYY-MM-DD’)

  • 时差：

    • 时差 (之后，现在为基准)：moment([2007, 0, 29]).fromNow(boolean); // 10 years ago
    • 时差 (之后，现在为基准)：moment([2017, 0, 29]).toNow(boolean); // in 10 years

  • 日历时间： moment().calendar();

  • 将时间转换成array： moment().toArray(); //[2017, 0, 15, 13, 24, 14, 95];
  • 将时间转换成object： moment(- ).toObject(); //{years: 2017, months: 0, date: 15, hours: 13, minutes: 25…}
  • 是否 Moment 对象: moment.isMoment(moment()); // true

  • 语言设置(国际化):

    • 全局： moment.locale(‘zh_cn’);
    • 局部： moment().locale(String);
